# 3D Models

Place your 3D models here in GLB or GLTF format.

## Supported Formats

- `.glb` - Binary glTF (recommended)
- `.gltf` - JSON-based glTF

## Default Model

The AR viewer looks for `model.glb` by default. If no model is found, a green rotating cube will be displayed as fallback.

## Recommended Sources for Free 3D Models

- [Sketchfab](https://sketchfab.com/features/free-3d-models) - Large collection, filter by "Downloadable"
- [poly.pizza](https://poly.pizza/) - Low-poly models, great for AR
- [Kenney Assets](https://kenney.nl/assets) - Game-ready assets
- [Google Poly Archive](https://poly.google.com/) - Via Internet Archive

## Tips

- Keep file size under 5MB for mobile performance
- Use compressed textures (KTX2 or WebP)
- Optimize geometry (reduce polygon count)
- Test on mobile devices
