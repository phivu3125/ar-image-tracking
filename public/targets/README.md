# Target Images

This folder contains compiled MindAR target files (`.mind` format).

## How to Create Target Files

1. Go to [MindAR Image Targets Compiler](https://hiukim.github.io/mind-ar-js-doc/tools/compile)
2. Upload your target image(s) - JPG or PNG
3. Click "Start" to compile
4. Download the generated `.mind` file
5. Place it in this folder as `targets.mind`

## Requirements for Good Target Images

- High contrast patterns
- Unique, non-repetitive features
- Avoid solid colors or simple gradients
- Minimum 300x300 pixels recommended
- Avoid reflective or transparent surfaces

## Demo Target

For testing, you can use any image with good features. Some options:
- Book covers
- Posters
- Business cards
- Product packaging

The AR experience will overlay 3D content when the camera detects the target image.
